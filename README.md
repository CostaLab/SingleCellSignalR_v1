# SingleCellSignalR

A R package to study Cell Signaling Using Single Cell RNAseq Data.
 
# Installation

From Rstudio type:
```
devtools::install_github(repo = "https://github.com/SCA-IRCM/SingleCellSignalR_v1", subdir = "SingleCellSignalR")    
```
